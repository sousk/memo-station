Firefox - Web の再発見

数々の賞に輝く Web ブラウザがさらに進化しました。Firefox は、ウイルスやスパイウェアなど様々な危険からあなたを守ります。使いやすさやパフォーマンスが向上し、プライバシー機能も強化されました。お気に入りや個人設定を簡単に取り込めるので、インストールしてすぐに使えます。Firefox で Web を最大限活用してください。

Firefox 1.5 のアップデートを公開

Firefox 1.5 の安定性、安全性を向上させた Firefox 1.5.0.1 を公開いたしました。現在 1.5 をお使いの方は、バックグラウンドで自動的に小さな差分更新モジュールがダウンロードされ、インストールの準備ができると確認ダイアログが表示されます。また、[ヘルプ] メニューから [ソフトウェアの更新を確認] を選択すれば、今すぐアップデートを行うことができます。この最新版は上記リンクからもダウンロードいただけます。詳細は リリースノート をご覧ください。

Firefox の機能と特長

最高の Web 体験を提供
安全で快適な Web の旅をお楽しみください。Firefox 1.5 は、誰にでも使いやすい直感的なユーザインターフェイスを備え、ウイルスやスパイウェア、ポップアップ広告からあなたを守ります。もちろん、インストールやお気に入りのインポートは簡単です。便利なタブブラウズ機能をはじめ、ライブブックマーク、組み込みの検索バーなど数々の機能を満載した Firefox は、かつてない Web 体験を提供します。
より高速な Web ブラウズを実現
履歴を戻る・進む際のページの読み込み速度が大幅にアップしました。また、ブラウザの核となるレンダリングエンジンに改良が加えられたことで、複雑なページをより正確に表示できるようになったほか、新しい Web 標準がサポートされ、全体的なパフォーマンスも向上しました。
自動アップデート
新しいソフトウェアアップデート機能によって、セキュリティパッチや新バージョンを簡単に入手できるようになりました。Firefox は小さな差分ファイルをバックグラウンドで自動的にダウンロードして、インストールの準備ができたことを知らせてくれます。
タブブラウズ機能
ひとつのウィンドウ内にいくつもページを開いておける タブブラウズ機能 を使えば、すばやく効率的な Web ブラウズが可能になります。ドラッグ＆ドロップでタブを並び替えて、関連するページをまとめることもできます。
ポップアップブロック機能
組み込みのポップアップブロック機能がさらに強化され、様々な方法で開かれるポップアップウィンドウやポップアンダー広告への対策を行いました。
検索エンジンの統合
ウィンドウ右上の検索バーから、人気の検索エンジンやショッピングサイトへいつでもアクセスできます。後から他のエンジンを追加することも可能です。
強固なセキュリティ
Firefox は、スパイウェアやワーム、ウイルスを寄せ付けず、安全な Web ブラウズを可能にします。開発者やセキュリティ専門家からなるコミュニティが 24 時間体制でセキュリティ問題を監視し、万一の際もすばやく対応します。
プライバシーデータの消去
新しい個人情報消去ツールがあなたのプライバシーを守ります。履歴、Cookie、フォーム入力データ、パスワードなどの個人情報をすべてワンクリックで消去できます。
ライブブックマーク
お気に入りのサイトやブログは欠かさずチェックしましょう。Web サイトから配信される最新情報で自動的に更新される ライブブックマーク をぜひご利用ください。
アクセシビリティ
Firefox 1.5 は、視覚障害や運動障害をお持ちの方を含め、すべての人に使いやすいナビゲーションを提供します。Firefox は、新たなグラフィック要素でも利用可能な、Web コンテンツの読み上げを可能にする DHTML をサポートした最初のブラウザとなりました。マウスクリックの代わりにキーボードを使った場合に問題となる、表計算データのようなドキュメントの移動に必要なタブ操作も減らしました。また、体に障害をお持ちの方がソフトウェアに容易にアクセスできるよう定めた米国政府の要件にブラウザとして初めて適合しました。
カスタマイズも自由自在
よく使うツールバーのボタンを選んで並べ替えたり、拡張機能をインストールして新たな機能を追加したり、多彩なテーマでデザインを楽しんだり - Firefox は自分のスタイルに合わせた使い方ができるのです。
次世代 Web を先取り
革新的な Web アプリケーションや Web サービスは、私たちにより豊かな Web 体験を提供してくれます。Firefox は、オープンな Web 標準をサポートすることで、こうした新しい Web ベースのツールを最大限活用できるようにしました。
